TracePlayer — отслеживание игроков

TracePlayer — это AMXX-плагин для CS 1.6, который хранит историю ников игроков, а также отображает информацию о банах в Steam и Fungun ECD.

Функциональность

  • Сохраняет все использованные игроком ники.
  • Отслеживает IP-адреса и время заходов.
  • Интеграция с Steam API — отображение статуса банов игрока
  • Интеграция с Fungun API — отображение статуса проверок на Fungun ECD
  • Просмотр информации о игроке прямо на сервере либо через веб-интерфейс

Как присоединиться

  1. Свяжитесь с автором, чтобы получить ваш уникальный API-ключ.
  2. Размесите на хостинге 2 файла player.php (необходим API-ключ, STEAM-API-ключ, Fungun-API-ключ) и player.css.
  3. Установите 2 AMXX-плагина: TracePlayer API для работы плагина необходимо установить модуль EasyHttp и прописать API-ключ; TracePlayer Core для работы плагина необходимо указать ссылку на файл player.php

После установки плагина и подключения к системе, информация о ваших игроках начнёт автоматически сохраняться и станет доступна через веб-интерфейс.

Условия

  • Плагин был создан для отслеживания игроков на HNS серверах. Если вы хотите установить всю систему отдельно, то свяжитесь с автором.
