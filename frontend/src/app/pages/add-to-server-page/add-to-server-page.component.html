TracePlayer — отслеживание игроков

TracePlayer — это AMXX-плагин для CS 1.6, который хранит историю ников игроков, а также отображает информацию о банах в Steam и Fungun ECD (в разработке).

Функциональность

  • Сохраняет все использованные игроком ники.
  • Отслеживает IP-адреса и время заходов.
  • Интеграция с Steam API — отображение статуса банов игрока
  • Интеграция с Fungun API — отображение статуса проверок на Fungun ECD (в разработке)
  • Просмотр информации о игроке прямо на сервере либо через веб-интерфейс

Как присоединиться

  1. Установите AMXX-плагин TracePlayer на ваш сервер. (Для работы плагина необходим модуль gRIP).
  2. Свяжитесь с автором проекта, чтобы получить ваш уникальный API-ключ.
  3. Добавьте ключ в настройки плагина — и всё готово!

После установки плагина и подключения к системе, информация о ваших игроках начнёт автоматически сохраняться и станет доступна через веб-интерфейс.

Условия

  • Плагин был создан для отслеживания игроков на HNS серверах
  • Если вы имеете свой HNS сервер то API-ключ вам будет доступен бесплатно
